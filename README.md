# Java_EnterManyObjectsInArrayList
Exercise
Write a Java program that has a structure that contains
students and faculty, each with a name attribute,

PIN. The student has an additional faculty number (no more
greater than 8 characters, otherwise the default value is 10101010) and

years, and for teachers to introduce a department and internship.
To compile ONE structure, which contains representatives of
students and teachers. Enter information in it and
write "logic" that:
• To be able to show the number of students after adults from
25 years and teachers with more than 30 years of experience;
• Go through the ordered structure and print in
console information for faculty and students.
